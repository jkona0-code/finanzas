Sincronizar con Google Sheets

Pega la URL de tu Google Apps Script Web App para sincronizar los datos. ¿Cómo configurarlo?

Pasos:
1. Crea una hoja de cálculo en Google Drive
2. Ve a Extensiones → Apps Script
3. Pega el código del archivo google-apps-script.gs (incluido en el repo)
4. Guarda y haz clic en Implementar → Nueva implementación
5. Tipo: Aplicación web | Acceso: Cualquier usuario
6. Copia la URL y pégala aquí
